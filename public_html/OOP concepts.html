Principles of Object-Oriented Programming


â€¢ OOP ( object oriented programing) ,In Python, object-oriented Programming (OOPs) is a programming paradigm that uses objects and classes in programming. It aims to implement real-world entities like inheritance, polymorphisms, encapsulation, etc. in the programming.


    Inheritance

  • Inheritance is the procedure in which one class inherits the attributes and methods of another class. The class whose properties are inherited are called Parent class. And the class that inherits the properties from the parent class is the Child class. The interesting thing is, along with the inherited properties and methods, a child class can have its own properties and methods.
  • In Calculator application, this method allows to use the fields and methods from previously existing classes. For example, the addition, subtraction, multiplication, and division classes can inherit the properties of the calculation class. Also, the calculation history class inherits properties from each calculation class. For example, the history class can inherit the properties of the addition and other class functions. This method helps to reduce the confusion and complexity of the program code.

    • Encapsulation

  • his puts restrictions on accessing variables and methods directly and can prevent the accidental modification of data.
  • In our calculator application, we use encapsulation by establishing classes with their own methods, variables, and functions. The history of calculations is kept in a list, and the history calculation class is provided with a clear history function. Encapsulation prevents a function class, such as the addition class, from clearing the history class's history. The clear history function is only useful for the class for which it was created.

    • Polymorphism

  • The word polymorphism means having many forms. In programming, polymorphism means the same function name being used for different types.
  • The get result() method in the calculator program is an interesting illustration of polymorphism. The addition, subtraction, multiplication, and division classes all have this function. The get result() function can be used for all of them, despite the fact that they are different classes. This is comparable to how the Python method len() can be used to represent a variety of data types, such as numbers, and lists.

    • Abstraction

  • It is the concept of object-oriented programming that shows only necessary attributes and hides unnecessary information. The main purpose of abstraction is hiding the unnecessary details from the users.
  • Abstraction is selecting data from a larger pool to show only relevant details of the object to the user. So in calculator application, It helps in reducing programming complexity and efforts. It is one of the most important concepts of OOPs.
